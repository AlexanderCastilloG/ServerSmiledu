Instalar las dependencias
```
npm install
```

Modificar el archivo keys.ts, para la conexión a la BD
```javascript
{
    host: "YOUR_PORT",
    user: "YOUR_USER",
    password: "YOUR_PASSWORD",
    database: "YOUR_DATABASE"
}
```

Compilar el código de TypeScript a JS
```
npm run build
```

Ejecutar la Aplicación
```
node build/index
```
